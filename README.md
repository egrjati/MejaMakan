# MejaMakan

Monorepo aplikasi MejaMakan.

## Struktur Project
- client: Frontend (Next.js)
- server: Backend + Prisma

## Cara Menjalankan

### Client
1. Masuk ke folder client
2. Install dependency: npm install
3. Jalankan: npm run dev

### Server
1. Masuk ke folder server
2. Install dependency: npm install
3. Jalankan sesuai script yang tersedia di package.json server

## Catatan
Pastikan konfigurasi environment sudah sesuai sebelum run aplikasi.
